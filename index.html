This is my second git push. Before this i one, i pushed one file and to learn how delete a repository i deleted it. and now i gonna note some commands that are important to navigate through in this file.

  • git init <-- this command is used to initialize git in a file
  • git add . <-- is used to add all files in git repository
  • git config --global user.email"YOUR EMAIL" <-- this is used to change ur git email address.
  • git config --global user.name"YOUR NAME" <-- this is used to change ur git user name.
  • git config -l <-- shows us user information and to escape that list press "w" and "q".
  • git remote add origin https://github.com/RajdipRudra/my-first-github-repo.git <-- this is where ur file is gonna be.
  • git push -u origin main <-- this command is used to send ur file in github and store it there.
